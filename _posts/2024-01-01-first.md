## 포인터란 무엇인가?

포인터(Pointer)는 메모리 주소를 저장하는 변수입니다. C 언어에서 포인터는 강력한 기능을 제공하지만, 그만큼 주의해서 사용해야 합니다. 포인터를 제대로 이해하면 동적 메모리 관리, 효율적인 배열 처리, 함수 간의 데이터 전달 등 다양한 프로그래밍 기법을 구현할 수 있습니다.

## 1. 포인터의 기본 개념

### 메모리 주소

컴퓨터는 데이터를 메모리에 저장하며, 각 메모리 위치는 고유한 주소를 가집니다. 포인터는 이러한 메모리 주소를 저장하는 변수입니다.

### 포인터 선언

포인터를 선언할 때는 변수 타입 앞에 `*` 기호를 사용합니다.

```c
int *ptr;
위 예제에서 ptr은 int 타입 변수가 저장된 메모리 주소를 저장할 수 있는 포인터입니다.

2. 포인터의 사용
변수의 주소 가져오기: & 연산자
& 연산자를 사용하여 변수의 메모리 주소를 가져올 수 있습니다.

c
코드 복사
#include <stdio.h>

int main() {
    int var = 10;
    int *ptr = &var;

    printf("변수 var의 값: %d\n", var);
    printf("포인터 ptr이 가리키는 주소: %p\n", ptr);
    printf("포인터 ptr을 통해 접근한 값: %d\n", *ptr);

    return 0;
}
출력 결과
csharp
코드 복사
변수 var의 값: 10
포인터 ptr이 가리키는 주소: 0x7ffee3bff5ac
포인터 ptr을 통해 접근한 값: 10
포인터를 통한 값 변경
포인터를 사용하면 변수의 값을 직접 변경할 수 있습니다.

c
코드 복사
#include <stdio.h>

int main() {
    int var = 20;
    int *ptr = &var;

    printf("변수 var의 초기 값: %d\n", var);

    *ptr = 30; // 포인터를 통해 var의 값 변경

    printf("포인터 ptr을 통해 변경한 var의 값: %d\n", var);

    return 0;
}
출력 결과
csharp
코드 복사
변수 var의 초기 값: 20
포인터 ptr을 통해 변경한 var의 값: 30
3. 포인터와 배열
배열의 이름은 배열의 첫 번째 요소의 메모리 주소를 가리키는 포인터로 해석될 수 있습니다.

c
코드 복사
#include <stdio.h>

int main() {
    int arr[5] = {1, 2, 3, 4, 5};
    int *ptr = arr; // 배열의 첫 번째 요소 주소

    for(int i = 0; i < 5; i++) {
        printf("arr[%d] = %d\n", i, *(ptr + i));
    }

    return 0;
}
출력 결과
css
코드 복사
arr[0] = 1
arr[1] = 2
arr[2] = 3
arr[3] = 4
arr[4] = 5
4. 포인터와 함수
포인터를 사용하면 함수에 변수를 참조로 전달할 수 있습니다. 이를 통해 함수 내에서 변수의 값을 변경할 수 있습니다.

예제: 두 변수의 값을 교환하는 함수
c
코드 복사
#include <stdio.h>

// 함수 선언: 포인터를 매개변수로 받음
void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int main() {
    int x = 5;
    int y = 10;

    printf("교환 전: x = %d, y = %d\n", x, y);

    // 포인터를 통해 함수에 변수의 주소 전달
    swap(&x, &y);

    printf("교환 후: x = %d, y = %d\n", x, y);

    return 0;
}
출력 결과
코드 복사
교환 전: x = 5, y = 10
교환 후: x = 10, y = 5
5. 동적 메모리 할당
포인터를 사용하면 런타임에 메모리를 동적으로 할당하고 해제할 수 있습니다. 이를 통해 필요한 만큼의 메모리를 효율적으로 사용할 수 있습니다.

예제: 동적 배열 할당
c
코드 복사
#include <stdio.h>
#include <stdlib.h>

int main() {
    int n;
    printf("배열의 크기를 입력하세요: ");
    scanf("%d", &n);

    // 동적 메모리 할당
    int *arr = (int *)malloc(n * sizeof(int));

    if(arr == NULL) {
        printf("메모리 할당 실패\n");
        return 1;
    }

    // 배열에 값 입력
    for(int i = 0; i < n; i++) {
        printf("arr[%d] = ", i);
        scanf("%d", &arr[i]);
    }

    // 배열 출력
    printf("입력한 배열: ");
    for(int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");

    // 메모리 해제
    free(arr);

    return 0;
}
사용 방법
프로그램 실행 후 배열의 크기를 입력합니다.
각 배열 요소에 값을 입력합니다.
입력한 배열이 출력됩니다.
6. 포인터의 주의사항
포인터는 강력하지만 잘못 사용하면 프로그램에 치명적인 오류를 초래할 수 있습니다. 다음 사항에 유의하세요.

초기화되지 않은 포인터: 초기화되지 않은 포인터를 사용하면 예기치 않은 동작이 발생할 수 있습니다.

c
코드 복사
int *ptr; // 초기화되지 않음
*ptr = 10; // 위험: 정의되지 않은 메모리 접근
메모리 누수: 동적 메모리를 할당한 후 해제하지 않으면 메모리 누수가 발생할 수 있습니다.

c
코드 복사
int *ptr = (int *)malloc(sizeof(int));
// 사용 후 반드시 free(ptr) 호출
잘못된 메모리 접근: 유효하지 않은 메모리 주소를 참조하면 프로그램이 비정상 종료될 수 있습니다.
